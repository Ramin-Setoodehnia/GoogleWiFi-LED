# Sync Google WiFi LEDs With Passwall2

## Fast Install

```bash
curl -sSL https://raw.githubusercontent.com/Ramin-Setoodehnia/GoogleWiFi-LED/refs/heads/main/install.sh -o /tmp/install.sh && chmod +x /tmp/install.sh && /tmp/install.sh
```
```bash
reboot
```
